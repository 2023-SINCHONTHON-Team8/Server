## 프로젝트 소개
밥메이트는 밥친구 찾기에 어려움을 겪고 있는 신촌 지역 대학생들을 위해 밥친구 추천을 제공하는 서비스입니다.
맵부심 단계, 민초•마라•고수 등 호불호, 모임의 분위기 등 자신의 선호와 맞는 밥친구를 추천해줍니다. 

밥친구 매칭 1회당 밥그릇 한공기가 적립됩니다.
밥그릇 50개 이상을 모아 슈퍼밥메 유저가 되어보세요!
